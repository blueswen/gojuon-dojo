# 五十音道場

專為學習日文五十音設計的網頁應用，透過羅馬字對應假名的練習，幫助快速熟記日文五十音。

功能：

1. 多樣練習模式：提供多種練習選項，包括平假名、片假名、濁音、半濁音、拗音，以及混合練習。
2. 即時發音提示：正確輸入後，自動播放對應的發音，強化聽覺記憶，提升學習效率。
3. 五十音圖學習：提供完整的五十音圖，包含平假名、片假名、濁音、半濁音與拗音，點擊任意假名，可聆聽正確發音。
